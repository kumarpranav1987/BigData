#####################       Sqoop Commands        ################################
Verify Sqoop : sqoop version

For Help : sqoop help (It will list all commands)

For help related to a particulat command :           
          sqoop <Command> --help
           Example : sqoop list-databases --help
           
1)List all databases  
sqoop list-databases --connect "jdbc:mysql://localhost" --username root --password password 
2)List all tables of a DB 
sqoop list-tables --connect "jdbc:mysql://localhost/movie_rating" --username root --password password 
3) Run a select query  
sqoop eval --connect "jdbc:mysql://localhost/movie_rating" --username root --password password --query "select * from movies"
